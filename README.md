Spring-React-Isomorphic-Example
==================================

Spring 의 ScriptTemplate을 활용한 React 예제 프로젝트 입니다.

# 요구사항
- JAVA 8+
- Spring Framework 4.2+
- Spring Boot 1.3+ 

# 실행방법
```
$ ./gradlew bootRun
```
